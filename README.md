# Chora
A JavaScript simulator of the [Patmos ISA](http://patmos.compute.dtu.dk/), developed by Marc Sun BÃ¸g & Simon Amtoft Pedersen as a part of their bachelor project at the Technical University of Denmark (DTU), advised by [Martin Schoeberl](https://www.imm.dtu.dk/~masca/) from DTU Compute. 

# Setup Chora
First install NodeJS and clone the repository from GitHub.
Then install dependencies in the root folder of the repository with
```
npm i 
```

## Run Chora as browser application
Open cmd and change scope to root folder of Chora and run
```
npm start
```

## Run JEST tests
Open cmd and change scope to root folder of Chora and run
```
npm run test
```
